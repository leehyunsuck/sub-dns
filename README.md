# NullDNS

NullDNS는 현재 운영 중인 무료 서브도메인 등록 서비스입니다.  

사용자는 별도의 비용 없이 자신만의 서브도메인을 생성하고 관리할 수 있으며, 간단한 절차로 빠르게 도메인을 배포할 수 있습니다.  

- **서비스 URL:** [https://nulldns.top](https://nulldns.top)  
- **주요 기능:**  
  - 무료 서브도메인 등록  
  - DNS 레코드 관리  
  - 간단하고 직관적인 사용자 인터페이스  

NullDNS를 통해 누구나 쉽게 인터넷 상에서 자신만의 도메인을 운영할 수 있습니다.
